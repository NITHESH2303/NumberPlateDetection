# License Plate Detection System

Detect and recognize vehicle license plates from images using YOLOv5 and EasyOCR.

## Features

- Real-time license plate detection
- Automatic text recognition (OCR)
- Multi-plate detection support
- High accuracy on various lighting conditions

## How to Use

1. Upload an image containing vehicles
2. Click "Submit"
3. View detected plates with bounding boxes and recognized text

## Technical Details

- **Detection Model**: YOLOv5
- **OCR Engine**: EasyOCR
- **Framework**: PyTorch
- **Interface**: Gradio
